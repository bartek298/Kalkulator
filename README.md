
# 🧮 Kalkulator C#

Prosta i funkcjonalna aplikacja kalkulatora zbudowana w środowisku **Visual Studio**. Idealny przykład implementacji logiki matematycznej w interfejsie graficznym (GUI).

---

## 🚀 Funkcje
* **Podstawowe operacje:** Dodawanie, odejmowanie, mnożenie i dzielenie.
* **Obsługa błędów:** Zabezpieczenie przed dzieleniem przez zero.
* **Intuicyjny interfejs:** Wygląd wzorowany na klasycznych kalkulatorach systemowych.

## 🛠️ Specyfikacja techniczna
* **Język:** C#

## 📖 Jak używać
1. Skonfiguruj środowisko **Visual Studio**.
2. Pobierz projekt i otwórz plik `.sln`.
3. Uruchom aplikację (`F5`).
4. Korzystaj z interfejsu przycisków, aby wykonywać obliczenia.
